# Imersão Dev com Google Gemini da Alura

Este projeto desenvolvido durante a 10ª imersão dev da alura, trata-se de um site que organiza e apresenta algumas obras de Clarice Lispector.
O usuário pode pesquisar livros, visualizar uma breve introdução e acessar resumos através da opção "saiba mais".
O foco foi manter uma interface direta, funcional e simples, aplicando todo o conhecimento adquirido durante toda a imersão.
Esse projeto junta a minha evolução em tecnologia e o meu gosto por literatura.
